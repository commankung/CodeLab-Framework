วิธีการใช้งาน phpmailer class

1.ให้ upload ไฟล์ทั้ง 3 ไฟล์ไปไว้ใน path ที่ต้องการส่งเมล์ครับ
    (ไฟล์ตัวที่จะทำหน้าที่ process ต้องอยู่ที่เดียวกันกับ 2 ไฟล์นี้คือ class.smtp.php และ class.phpmailer.php)
2. ไฟล์ sample.php เป็นตัวอย่าง code ครับ จะมีคอมเมนต์อธิบายแต่ละตัวแปรอยู่แล้วนะครับ
3. ในกรณีที่ต้องการส่งหลาย ๆ ครั้งสามารถทำ for loop หรือใช้ิวิธี cc ได้ครับ
